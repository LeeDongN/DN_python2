단, x좌표, y좌표 구할 때 처음 나오는 값과 제일 마지막에 나오는 값을 저장해야 함.

1. 기준이 되는 x, y 값 구하기  OK
2. 기준 값 x, y를 fix해 놓고 [y, pixel] 혹은 [x, pixel] 구하기
      (함수 호출 여러번 하지 않게 바꾸기)
3. 인접한 pixel값들의 차를 구해 자를 곳 구하기
  3-1. 첫번째 값이 꽉 차있을 수 있으니 첫번째 픽셀이 0이 아니면 바로 컷
    (y값이 꽉 차 있을 경우 -> x를 먼저 구해서 [0,0]부터 [0, n] 중에 차이가 커지는
   값이 있으면 잘라버리자
